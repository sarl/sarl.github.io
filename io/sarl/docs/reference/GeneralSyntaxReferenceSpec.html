General Syntax Reference

  • Spec
  • Source

This document describes the general syntax of the SARL Language. While we will use the agent definition, it is also valid for other concepts. Please see the specific Reference documentation for details.

  • Variable definition

    agent A {
  //Variable defnition
  var name : String
  
  //final variable definition - It can not be reassigned
  val string : String = "string value"
  
  //Infered type to String
  var lastname = "Lastname"
}

  • Java compatibility

    agent A {
  //Variable defnition
  var date : java.util.Date = new java.util.Date()
  
  //final variable definition - It can not be reassigned
  val otherDate : java.util.Date = new java.util.Date()
  
  //Infered type to java.lang.String
  var lastname = "Lastname"
}
