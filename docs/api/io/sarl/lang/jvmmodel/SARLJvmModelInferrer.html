  • java.lang.Object

org.eclipse.xtext.xbase.jvmmodel.IJvmModelInferrer
@javax.inject.Singleton
class SARLJvmModelInferrer
extends java.lang.Object
Infers a JVM model from the source model.

The JVM model should contain all elements that would appear in the Java code which is generated from the source model. Other models link against the JVM model rather than the source model.

The roles of the different generation tools are:
  • io.sarl.lang.jvmmodel.SARLJvmModelInferrer: Generating the expected Java Ecore model from the SARL Ecore model.
