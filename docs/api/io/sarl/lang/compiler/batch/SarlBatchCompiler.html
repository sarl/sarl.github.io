io.sarl.lang.compiler.batch
Class SarlBatchCompiler
  • java.lang.Object
    • SarlBatchCompiler
class SarlBatchCompiler
extends java.lang.Object
The compiler from SARL that could be used for batch tasks (Maven, CLI).

This compiler is inspired by the Xtend batch compiler.
